# Maximum Subarray

## Problem

Given an integer array `nums`, find the **subarray** with the largest sum, and return its sum.

---

## Examples

### Example 1
**Input**: `nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]`  
**Output**: `6`  
**Explanation**: The subarray `[4, -1, 2, 1]` has the largest sum `6`.

### Example 2
**Input**: `nums = [1]`  
**Output**: `1`  
**Explanation**: The subarray `[1]` has the largest sum `1`.

### Example 3
**Input**: `nums = [5, 4, -1, 7, 8]`  
**Output**: `23`  
**Explanation**: The subarray `[5, 4, -1, 7, 8]` has the largest sum `23`.

---

## Approach

### Dynamic Programming (Kadaneâ€™s Algorithm)
- **Idea**: Maintain a running sum (`current_sum`) and a maximum sum (`max_sum`).  
- At each step:
  1. Update `current_sum` to include the current number.
  2. If `current_sum` becomes negative, reset it to the current number.
  3. Update `max_sum` if `current_sum` is greater.
- **Complexity**:
  - **Time Complexity**: `O(n)` (Single pass through the array)
  - **Space Complexity**: `O(1)` (No extra space)

---

## Solution

```python
class Solution:
    def maxSubArray(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        max_sum = float('-inf')
        current_sum = 0
        
        for num in nums:
            current_sum = max(num, current_sum + num)
            max_sum = max(max_sum, current_sum)
        
        return max_sum
